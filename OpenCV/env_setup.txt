This document introduce how to setup opencv develoment environment on Ubuntu-16.04:

1) Install cmake:
	sudo apt-get install cmake	(cmd)
	sudo apt-get install cmake-gui	(UI)

2) Install OpenCV:
    For detail, please refer to:
	http://docs.opencv.org/master/d7/d9f/tutorial_linux_install.html
    --- Get Getting the Cutting-edge OpenCV from the Git Repository or download tar and extract
	git clone https://github.com/opencv/opencv.git
	git clone https://github.com/opencv/opencv_contrib.git
    
    --- Install required packages:
	[compiler] sudo apt-get install build-essential
	[required] sudo apt-get install cmake git libgtk2.0-dev pkg-config libavcodec-dev libavformat-dev libswscale-dev
	[qt]       sudo apt‐get install libqt4‐dev libvtk5‐qt4‐dev
	[optional] sudo apt-get install python-dev python-numpy libtbb2 libtbb-dev libjpeg-dev libpng-dev libtiff-dev libjasper-dev libdc1394-22-dev
    
    --- Building OpenCV from Source Using CMake
	a) Create a temporary directory, which we denote as <cmake_build_dir>, where you want to put the generated Makefiles, project files as well the object files and output binaries and enter there.
	For example:
		cd ~/opencv
		mkdir build
		cd build
	b) Configuring. Run cmake [<some optional parameters>] <path to the OpenCV source directory> 
	For example:
		sudo cmake -D CMAKE_BUILD_TYPE=Release -D CMAKE_INSTALL_PREFIX=/usr/local ..
	c) Build:
		sudo make
	d) Install:
		sudo make install	

3) Install Qt dependency:
    Qt install: https://www.qt.io/download-open-source/?hsCtaTracking=f977210e-de67-475f-a32b-65cec207fd03%7Cd62710cd-e1db-46aa-8d4d-2f1c1ffdacea
    
    For example:
        sudo apt-get install libqt4-designer libqt4-opengl libqt4-svg libqtgui4 libqtwebkit4

4) Install boost:
        a) Get the version of Boost that you require
                wget -O boost_1_55_0.tar.gz http://sourceforge.net/projects/boost/files/boost/1.55.0/boost_1_55_0.tar.gz/download
                tar xzvf boost_1_55_0.tar.gz
                cd boost_1_55_0/
        b) Boost's bootstrap setup:
                ./bootstrap.sh --prefix=/usr/local
        c) Then build it with:
                ./b2
        d) eventually install it:
                sudo ./b2 install
